DALY工具箱 隐私权政策

最后更新日期：2026年6月26日

感谢您使用 DALY工具箱（以下简称"本扩展"）。我们重视您的隐私，本隐私权政策说明本扩展如何收集、使用和保护您的信息。

1. 数据收集

本扩展在您使用期间会收集以下类型的数据：

  • 网页内容数据：当您在电商平台页面上使用采集功能时，本扩展会读取当前页面的商品信息（如商品标题、价格、图片等），仅用于帮助您完成数据采集操作。
  • 本地存储数据：本扩展使用 Chrome Storage API 在本地保存您的偏好设置和操作记录，这些数据仅存储在您的本地设备上，不会上传至任何服务器。

2. 数据使用

我们收集的数据仅用于以下目的：

  • 提供商品数据采集和下载功能
  • 保存用户设置以改善使用体验
  • 在浏览器本地完成所有数据处理

3. 数据共享

本扩展不会将您的任何个人数据共享、出售或转让给第三方。

4. 数据存储

本扩展的所有数据均存储在您的本地浏览器中（通过 Chrome Storage API）。本扩展不设立或使用任何远程服务器，不会将数据上传至任何外部服务。

5. 权限说明

本扩展需要以下 Chrome 权限：

  • activeTab / scripting：用于在当前页面注入脚本以读取商品信息
  • downloads：用于将采集的数据下载到本地
  • webRequest：用于在特定电商平台页面拦截和读取网络请求数据
  • storage：用于在本地存储用户设置
  • host_permissions：仅在用户访问特定电商网站时生效，用于读取页面内容

6. 数据安全

由于本扩展不使用远程服务器，所有数据均存储在您的本地设备上。我们建议您定期清理浏览器数据以保护您的隐私。

7. 用户控制

您可以随时通过以下方式管理您的数据：

  • 在 Chrome 扩展管理页面中清除本扩展的存储数据
  • 卸载本扩展将自动删除所有本地存储数据

8. 政策更新

我们可能会不时更新本隐私权政策。更新后的政策将在此页面发布，并更新"最后更新日期"。
