## 安装说明

### 方法一：从 Chrome 网上应用店安装（推荐）

1. 访问 [DALY工具箱官网](https://daly.433345.xyz/)
2. 点击"安装扩展"按钮
3. 在 Chrome 应用店中确认安装

### 方法二：从源码安装（开发者模式）

1. 下载本项目源码
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹
6. 扩展安装完成

## 使用方法

### 电商平台

1. 打开对应平台页面（如淘宝商品页、拼多多 MMS 后台等）
2. 页面会自动加载 DALY 工具箱悬浮窗
3. 点击悬浮窗查看实时数据
4. 使用"下载媒体"功能批量下载商品图片/视频

### 自媒体平台

1. 打开抖音视频页、小红书笔记页或快手视频页
2. 点击页面右侧的 DALY 工具箱图标
3. 选择"下载视频"或"下载图文"
4. 在弹出面板中选择要下载的资源
5. 点击"一键打包下载"或单独下载
